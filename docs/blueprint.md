# **App Name**: Inclusive Voyager

## Core Features:

- Personalized Trip Planner: Generates tailored travel itineraries based on disability type, budget, and destination preferences.
- Smart Travel Assistant: A chatbot providing real-time support and information before and during trips.
- Accessible Maps: Maps highlighting accessible routes, facilities, and points of interest at tourist locations.
- Rights and Support Directory: A directory of local support organizations and legal rights information for travelers with disabilities at each location; information is surfaced by the AI as a tool.
- Solidarity Host Connection: Connects travelers with local volunteers or 'solidarity hosts' for assistance and companionship.
- Sign Language Translation: Automatic translation of spoken content into sign language for users with hearing impairments.
- Digital Health Passport: Stores user disability information and shares this data with hotels/airlines.
- Emergency System: Sends the location and user status via the Firebase database to a list of emergency contacts when activated.

## Style Guidelines:

- Primary color: Soft teal (#70A1A9) to create a sense of calm and serenity, reflecting a stress-free travel experience.
- Background color: Light gray (#F0F4F5), a desaturated shade of teal, providing a neutral and accessible backdrop.
- Accent color: Muted purple (#8E77A1), an analogous color that is lower in saturation and brightness, is used for interactive elements and important notifications, ensuring clear visibility.
- Body: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for body text. Headlines: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, clear icons representing various accessibility features and travel-related actions.
- A clean, spacious layout with a strong degree of control over contrast levels
- Gentle transitions and subtle animations to enhance user experience without causing distraction.
